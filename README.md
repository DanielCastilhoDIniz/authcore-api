# AuthCore API

AuthCore é uma API back-end desenvolvida em Python com Django e Django REST Framework, projetada para servir como núcleo de autenticação, gerenciamento de usuários e controle de permissões em sistemas web.

O projeto tem como objetivo fornecer uma base sólida, escalável e reutilizável para aplicações educacionais e institucionais, seguindo boas práticas de engenharia de software, organização de código, segurança e documentação.

Esta API está sendo construída com foco em clareza arquitetural, qualidade de código e evolução progressiva, servindo também como projeto de portfólio profissional.

## Stack
- Python 3.11+
- Django
- Django REST Framework
- PostgreSQL
